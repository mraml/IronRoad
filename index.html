Iron Road runs in the browser like any other web game, but this folder is the source project — not the hosted site yet.

Developers: run npm install then npm run dev and open the http://localhost:… URL (double‑clicking this file will not work).

Players / any device: someone runs npm run build, uploads the dist folder to Netlify / Vercel / any web host, then you open the https://… link on your phone or PC — no Node required on the device.

See README.md → “Play on any device”.
